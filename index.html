INSTRUCTION

  • Turn on calculator, press C/on.
  • Turn off calculator, press off.
  • Clear calculator, but not memory, press C/on twice.
  • Clear entry, overflow, or error, press C/on once.
  • Change sign of display, press +/– once.
  • Add contents of display to memory, press M+.
  • Recall memory to display, press RM once.
  • Clear memory, press RM twice.

ABOUT

The "Retro Calculator Simulator" was created with inspiration from "Litronix 2140" pocket calculator. The actual functionality and outlook may differ from original, because I didn't posses the physical calculator.

Vectorial image was created using the photo by Hans Bloemen from calculatormuseum.nl

As page background an image by GarryKillian from freepick.com was used.

The calculator is drawn using open source software "Inkscape", the app is created using HTML, CSS and JavaScript.

You can take a look at the code at projects GitHub.
